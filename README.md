## Deskripsi
Proyek ini bertujuan untuk memprediksi **kualitas anggur (wine quality)** berdasarkan fitur-fitur kimiawi menggunakan model **Random Forest Classifier**.

## Tahapan Analisis
1. Persiapan dan pembersihan data  
2. Normalisasi fitur (StandardScaler)  
3. Pembuatan dan pelatihan model Random Forest  
4. Evaluasi model dengan akurasi dan confusion matrix  
5. Prediksi kualitas anggur pada data testing  
6. Penyimpanan hasil dalam `hasilprediksi_028.csv`

## Hasil
- Akurasi Validasi: 0.6105  
- Akurasi Cross Validation: 0.6371
